SRAM，可讀可寫，拔掉電源後裡頭的東西就會消失不見。負責存放靜態資料、堆積（heap）、堆疊（stack），所以是程式裡建立變數進行操作的地方。靜態資料，若不指定初始值則為0，若有則從Flash拷貝到SRAM。堆積，動態配置的記憶體空間，譬如malloc。堆疊，存放函式呼叫時所需東西、區域變數。堆積與堆疊會從SRAM的兩端逐漸成長
